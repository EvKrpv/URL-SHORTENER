URL Shortener
Сервис сокращения ссылок на Go с использованием PostgreSQL в качестве основного хранилища и Redis для кэширования. При создании короткой ссылки генерируется уникальный 10-символьный код с помощью криптостойкого генератора crypto/rand, после чего запись сохраняется в базу данных с заданным временем жизни (по умолчанию 30 дней). При переходе по короткой ссылке запрос сначала обрабатывается через Redis для минимизации нагрузки на PostgreSQL, а при отсутствии в кэше выполняется запрос к базе с последующим кэшированием результата.

Архитектура проекта построена по принципу разделения ответственности: handlers отвечают за HTTP-взаимодействие, services содержат бизнес-логику и работу с кэшем, repositories инкапсулируют операции с базой данных. Приложение использует gin в качестве HTTP-фреймворка, pgx для эффективной работы с PostgreSQL и go-redis для взаимодействия с Redis. Docker-контейнеризация позволяет быстро развернуть весь стек командой docker-compose up -d.

Проект находится в активной разработке. На данный момент реализованы основные эндпоинты: POST /shorten для создания ссылки и GET /:shortCode для редиректа. В планах — добавление эндпоинта /info для получения статистики, подсчет количества переходов, автоматическая очистка просроченных ссылок, поддержка кастомных алиасов и веб-интерфейс для удобного управления ссылками.
